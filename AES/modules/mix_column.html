// GF(2^8) ÏÇ¨Ï†Ñ Í≥ÑÏÇ∞ ÌÖåÏù¥Î∏î Ï∞∏Ï°∞
#define gmult(a,b) gmult_aes[a][b]

void mix_columns(uint8_t *state) {
    uint8_t a[] = {0x02, 0x01, 0x01, 0x03};
    uint8_t col[4], res[4];

    for (int j = 0; j < 4; j++) {
        for (int i = 0; i < 4; i++) col[i] = state[4*i+j];

        // ÌÖåÏù¥Î∏î Lookup Î∞è XOR Ïó∞ÏÇ∞
        
        // Row 0: {02, 03, 01, 01}
        res[0] = gmult(a[0],col[0]) ^ gmult(a[3],col[1]) ^ 
                 gmult(a[2],col[2]) ^ gmult(a[1],col[3]);

        // Row 1: {01, 02, 03, 01}
        res[1] = gmult(a[1],col[0]) ^ gmult(a[0],col[1]) ^ 
                 gmult(a[3],col[2]) ^ gmult(a[2],col[3]);

        // Row 2: {01, 01, 02, 03}
        res[2] = gmult(a[2],col[0]) ^ gmult(a[1],col[1]) ^ 
                 gmult(a[0],col[2]) ^ gmult(a[3],col[3]);

        // Row 3: {03, 01, 01, 02}
        res[3] = gmult(a[3],col[0]) ^ gmult(a[2],col[1]) ^ 
                 gmult(a[1],col[2]) ^ gmult(a[0],col[3]);

        for (int i = 0; i < 4; i++) state[4*i+j] = res[i];
    }
}
